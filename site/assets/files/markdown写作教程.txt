《PromiseWikki 写作速查表》**，你可以直接复制到你的 变大变粗)

标题决定了你右边那个“目录”长什么样。

* `#` 是一级标题（最大的，通常只用在文章最开头）。
* `##` 是二级标题（章节名）。
* `###` 是三级标题（小点）。

**代码：**

```markdown
# 我是最大的标题
## 我是二级标题
### 我是三级标题
#### 我是四级标题

```

---

### 2. 常用装饰 (加粗、斜体)

* **加粗**：用两个星号包起来。
* *斜体*：用一个星号包起来。
* ~~删除线~~：用两个波浪号包起来。

**代码：**

```markdown
这句话里**这个词很关键**。
这句话里*这个词是斜的*。
~~这句话我想收回~~。

```

⚠️ **注意**：星号和文字之间**不能有空格**！

---

### 3. 列表 (让条理更清晰)

**无序列表** (圆点)：用 `*` 或 `-` 加一个空格。
**有序列表** (数字)：用 `1.` 加一个空格。

**代码：**

```markdown
* 买苹果
* 买香蕉
    * (按一下 Tab 键可以缩进) 买进口香蕉

1. 第一步：打开冰箱
2. 第二步：把大象装进去

```

---

### 4. 链接与图片 (最容易错的地方！)

这里一定要**切回英文输入法**！

* **链接**：`[显示的文字](链接地址)`
* **图片**：`![图片描述](图片路径)` (前面多一个感叹号！)

**代码：**

```markdown
[去百度看看](https://www.baidu.com)

![我的猫](assets/images/cat.jpg)

```

---

### 5. 代码块 (程序员必备)

用三个反引号 ````` 包起来。如果在第一行写上 `python`，代码还会自动变色（高亮）。

**代码：**

```markdown
下面是我的 Python 代码：

```python
print("Hello World")
a = 1 + 1
``` (这里记得换行再闭合)

```

---

### 6. 引用 (像回帖一样)

用 `>` 开头。

**代码：**

```markdown
> 鲁迅曾经说过：在这个网站写 Markdown 真的很开心。

```

### ✨ 7. 你的“专属魔法” (MkDocs 特效)

这些是你配置了插件后才能用的高级功能！

#### 🎨 彩色提示框 (Admonitions)

一定要注意：**冒号后面要换行，并且里面的内容要缩进 (按 Tab)！**

**代码：**

```markdown
!!! note "普通的笔记"
    这是一个蓝色的笔记框。

!!! tip "小技巧"
    这是一个绿色的提示框。

!!! warning "警告"
    这是一个黄色的警告框。

!!! danger "危险"
    这是一个红色的危险框。
🟢 绿色系 (表示成功/推荐)

!!! success "成功"

!!! tip "技巧" (这个你用过，最常用！)

🟡 黄色系 (表示提醒)

!!! warning "警告"

!!! attention "注意"

🔴 红色系 (表示危险/错误)

!!! danger "危险"

!!! failure "失败"

!!! bug "Bug"

🔵 蓝色/青色系 (表示信息)

!!! info "信息"

!!! note "笔记"

🟣 紫色/深色系 (表示提问/引用)

!!! question "提问" (这个图标是个问号 ❓，很好看！)

!!! quote "引用"



Copied from: Google Gemini - <https://gemini.google.com/app/ae877ab689398e48?utm_source=app_launcher&utm_medium=owned&utm_campaign=base_all>

```

#### 🔘 粉色按钮

在链接后面紧贴着加上 `{ .md-button }`。

**代码：**

```markdown
[点我下载](assets/files/game.zip){ .md-button }

[点我下载 (实心色)](assets/files/game.zip){ .md-button .md-button--primary }

```
